Markdown使用介绍

2022-03-13 评论

一.创建文章

进入hexo的文件夹中

hexo n "要取的文章名字"

不要用管理员进行创建 否则没有权限编辑不了文章 这个问题困扰我好久

二.编写文章

(1)标题

#一级标题

##二级标题

……

######六级标题

Typora 快捷键：

一级标题：*（⌘ + 1）*

二级标题：*（⌘ + 2）*

三级标题：*（⌘ + 3）*

四级标题：*（⌘ + 4）*

…

六级标题：*（⌘+6）*

(2)粗体、斜体、删除线和下划线

斜体
粗体
加粗斜体
删除线

下划线

*斜体* **粗体** ***加粗斜体*** ~~删除线~~

Typora 快捷键：

（⌘ + i）：斜体

（⌘ + b）：粗体

（^ + ⇧ + `）：删除线

（⌘ + u）下划线

(3)引用块

>文字引用

这个就是一个引用

Typora 快捷键： （⌥ + ⌘ + q）

(4)代码块

Typora 快捷键：

（⌘ + ⌥ + c）

(5) 公式块

（⌘ + ⌥ + B）

(6) 分割线

（— + enter 或 ⌘ + ⌥ + -）

(7) 列表

Typora 快捷键：

有序列表项：（⌥ + ⌘ + o）

无序列表项：（⌥ + ⌘ + u）

任务列表：（⌥ + ⌘ + x）

(8) 表格

Typora 快捷键： (⌘ + ⌥ + T）

(9)超链接

快捷键：（⌘ + k）

(10)插入图片

（鼠标直接拖拽）

hexo博客

(11)文章跳转

跳转至文章开头（⌘ + 向上箭头）
跳转至文章结尾（⌘ + 向下箭头）

(12)选中英文单词/中文单词

选中英文单词或连续的中文 （⌘ + d）
选中光标左边英文单词/中文 （⌘+⌥+左方向）
选中光标左边英文单词/中文 （⌘+⌥+右方向）

(13)按行选中

（⌘ + l）

(14)查找替换

（⌘ + f）

(15)插入表情

mac输入 *⌃ + ⌘ + space* 弹出符号对话框

(16)新建文件

（⌘ + n）

(17)显示和隐藏侧边栏

（⇧ + ⌘ + L）

(18)标题居中

支持html语言 h1表示一级标题，h6表示六级标题

(19)文本居中

从网上查的文章结合自己使用的写出来的 日后还有补充 方便自己复习看
