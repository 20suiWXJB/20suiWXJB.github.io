java.util.List;与 java.awt.List; 的区别

  1. java.util.*与java.awt.都是java的标准库包，其中表示java.util和java.awt包里的所有类。
  2. java.util.*包含的是一些工具类，如集合类中List、Map、HashMap、Set等.
  3. java.awt .*则封装的是和图形绘制相关的类，如点Point、线Line等。
